# Chords

This is a tool to learn music theory.

## Features

- Display scale

    - [x] Major scale

- Display chords

    - [x] Major

    - [x] Minor

    - [x] Sus2

    - [x] Sus4

    - [x] Major seventh

    - [x] Dominant seventh

    - [x] Minor seventh

    - [x] Minor major seventh

    - [x] Half-diminished seventh

## Run

```
go run main.go
```
